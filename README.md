# AI Interview Preparation Assistant

## Project Description
AI Interview Preparation Assistant is a web application developed using Python and Streamlit. It helps users prepare for interviews by generating interview questions, evaluating answers, providing scores, strengths, weaknesses, and improvement suggestions.

## Features
- Select Job Role
- Generate Interview Questions
- Answer Evaluation
- Score Generation
- Strength and Weakness Analysis
- Improvement Suggestions

## Technologies Used
- Python
- Streamlit
- Hugging Face
- GitHub

## How to Run
1. Install Python
2. Install Streamlit
3. Run:
   streamlit run app.py
